
# node.js express 项目模板

这是一个 node.js express 项目模板

## 启动项目

```bash
git clone git@github.com:cjm0/express.git

yarn 

npm run dev

```

## 说明

`node-dev` 监听文件，当有修改后自动去重启nodejs进程

`nodemon` 监视 node.js 应用程序中的任何更改并自动重启服务

`brew services start mongodb` 启动 mongodb

`npm run dev` 本地启动

`npm run prod` 线上启动
